# NIST-KBKDFs
A .NET implementation of the [NIST SP 800-56Cr2](https://csrc.nist.gov/pubs/sp/800/56/c/r2/final) and [NIST SP 800-108r1](https://csrc.nist.gov/pubs/sp/800/108/r1/final) KDFs.
